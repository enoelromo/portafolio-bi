# Seguridad y datos sensibles

- Este repo **no** contiene credenciales ni direcciones reales.
- Usa *placeholders* como `<SRC_HOST>`, `<SRC_DB>`, `<SRC_USER>`, `<SRC_PASSWORD>`, `<SRC_PORT>`.
- No subas `.env`, respaldos o dumps con datos reales.
- Anonimiza datos (Cliente 001, Producto 001…) y difumina secretos en capturas.
