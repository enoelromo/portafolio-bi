# Security and Sensitive Data

- This repo **does not** contain credentials or real addresses.
- It uses *placeholders* such as `<SRC_HOST>`, `<SRC_DB>`, `<SRC_USER>`, `<SRC_PASSWORD>`, `<SRC_PORT>`.
- Do not commit `.env` files, backups, or dumps with real data.
- Anonymize data (Customer 001, Product 001...) and blur secrets in screenshots.
